# ⭐ Hourse 

‣ A Hourse é um site de entretenimento de mini-games de 2 jogadores,<br>
foi criado para até duas pessoas poderem jogador, se divertir<br>
e até mesmo apostar moeda fictícia dentro do site.

# 📌 Objetivo

‣ Hourse tem como objetivo proporcionar diversão para quem for utiliza-lo,<br>
com alguns mini-games de até 2 jogadores os jogadores podem disputar quem<br>
vai adquirir o valor total da aposta.

# 💡 De onde surgiu

‣ A ideia de criar este site veio por conta da matéria de educação financeira,<br>
que a professora criou moedas fictícia para a escola utilizar. Então criei este site<br>
para eu ter como aprendizado na programação e na educação financeira.

# 🧠 Aprendizados

▸ <strong>Criação de mini-games com JS</strong>;<br>
▸ <strong>UX intuitiva</strong>;<br>
▸ <strong>Integração com GoogleForms</strong>.
