# ⭐ Hourse 

A Hourse é um site de entretenimento de mini-games de 2 jogadores,<br>
foi criado para até duas pessoas poderem jogador, se divertir<br>
e até mesmo apostar moeda fictícia dentro do site.

# 📌 Objetivo

Hourse tem como objetivo proporcionar diversão para quem for utiliza-lo,<br>
com alguns mini-games de até 2 jogadores os jogadores podem disputar quem<br>
vai adquirir o valor total da aposta.
